# 3D Video with Python using Matplotlib

This project demonstrates how to create a **3D video** using Python and Matplotlib. The video is generated as a GIF by rotating a 3D plot and combining multiple frames.

---

## Features

- Generate 3D contour plots
- Rotate the plot from 0° to 360°
- Export all frames and create a looping GIF
- Fully Python-based solution
